# 任务栏（全自动流水线版）

只有被写在任务栏里的、且派给你的任务才能做。未在栏内或未派给你的，一律不做。

## 规则与状态流转（自动化标识）
- **[待办 (Todo)]**：由“规则助手”根据“需求解读 Agent”的产出自动新增。
- **[进行中 (In Progress)]**：由“前端/后端 Agent”在领活后自动更新。
- **[待验收 (Pending Review)]**：由“前端/后端 Agent”在提交 PR 并更新 STATUS.md 后自动更新。
- **[已完成 (Done)]**：由“人”在验收通过并合并分支后手动（或指令）更新。
