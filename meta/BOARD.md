# 任务栏（简化版，无表格）

只有被写在任务栏里的、且派给你的任务才能做。未在栏内或未派给你的，一律不做。

## 规则与状态流转
- **添加规则**：需求解读产出后，人读完并确认没问题，才把任务加入「当前任务栏」。
- **状态定义**：
  - **待办 (Todo)**：任务已定义，尚未开始。
  - **进行中 (In Progress)**：Agent 已领任务并在开发中。
  - **待验收 (Pending Review)**：Agent 已提交 PR 并更新了 STATUS.md。
  - **已完成 (Done)**：人验收通过并合并分支。
- **更新权限**：人审完需求解读产出后由人更新。前端/后端 Agent 只读，不修改本文件。
