# TASKS.md — 任务板（并行开发用）

## 使用规则（所有 agent 必须遵守）
- 每个任务必须在独立分支完成：feat/<task-id>-<short-name>
- 每个任务只能修改「允许改动范围」内的文件；超出范围必须先停下并说明原因
- 完成后必须在 STATUS.md 追加一条记录（按模板）
